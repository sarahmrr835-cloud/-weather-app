# 🌤️ Weather App

## 📌 Description
A responsive weather application that provides real-time weather information for any city using a weather API.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API

---

## ✨ Features

- Search weather by city
- Real-time weather data
- Temperature in Celsius
- Humidity information
- Wind speed
- Responsive design
- Error handling
- Clean UI


---

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/weather-app.git
```

2. Open the project folder.

3. Add your API key.

4. Run `index.html`.

---

## 📖 Usage

- Enter a city name.
- Click Search.
- View the weather information.
